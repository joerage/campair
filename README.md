## 🎮 What is Soc Ops?

Soc Ops is a **Social Bingo game** for in-person mixers — find people who match the prompts on your card and get 5 in a row to win!

But it's also more than a game. It's the **learning platform** for this hands-on workshop. You'll use VS Code's Agent Mode with GitHub Copilot to transform a working-but-plain FastAPI app into something genuinely great — practicing the same agentic workflows used by professional developers every day.

```
Player joins → Gets a unique 5×5 bingo card → Finds matching people → First to BINGO wins!
```

## ✨ What You'll Build

By the end of this workshop you'll have shipped:

- 🎨 **A redesigned UI** — crafted through AI-assisted design iteration, not default templates

- 🕹️ **New game modes** — Scavenger Hunt and Card Shuffle, built test-first
- ⚙️ **A fully wired codebase** — with context engineering that makes AI genuinely useful

## ✅ Prerequisites

- **VS Code v1.107+** (no pending updates)
- **GitHub Copilot** — Free, Pro, Business, or Enterprise
  > ⚠️ Free-tier users: Cloud Agents aren't available on free plans. Alternative instructions are provided throughout.
- **Git**, **Python 3.13+**, and **[uv](https://docs.astral.sh/uv/)**
- Chat panel open and Agent Mode ready

> 💡 Use the **DevContainer** for a zero-config environment — no local installs required!

